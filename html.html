File Management

Make sure that your files are stored in a logical manner where they are accessible and organized. Stick with one naming convention to avoid confusion. Try to find a way to be both descriptive and brief.

Template

<!DOCTYPE html> <html lang="en"> <head> <title></title> <meta charset="utf-8"> </head> <body> <header> </header> <nav> </nav> <main> </main> <footer> </footer> </body> </html>

Semantic Elements

The semantic elements are organizational elements within the body element. The header is an element where you would put important information you want to appear at the top of your webpage. It is different from the head, because you don't need to include the title and meta tags. The nav is your navigation bar, which usually consists of links to other pages in your website or even sections below on the same webpage. The main is the main content that you want to appear on your webpage. The footer generally contains contact info, dates updated or published, and any copyright info that you wish to display.

Comments and Contents

To make a comment in HTML, use <!--comment-->. Comments are used to add info to your code without it showing up in the actual webpage.

Symbol Entities are special characters which can be put into a website using special codes. For example, since the less than sign is a reserved character (meaning that typing it could cause a text editor to read it as part of a tag and become confused), it is usually typed using an entity name to prevent any errors in code. The entity name for this symbol is & lt; and the entity name for a nonbreaking space is &nbsp; which looks like this  . You can also create these characters by using their entity number. For example, this star ✩ can be created by using its entity number of &#10025;. Most elements have both an entity name and number.

When creating the main content of your page, there are many helpful tags. Here is a list to get started, shown without the brackets.

  • p: paragraph tag
  • h1: heading 1
  • h2: heading 2
  • h3: heading 3
  • ul: unordered/bulleted list
  • ol: ordered/numbered list
  • li: list element
  • dl: description list, a specific type of list used for terms and their descriptions
  • dt: description list term
  • dd: description list definition
  • br: creates a line break
  • hr: creates a solid line, called a thematic break
  • div: used to create a container element that is helpful for labeling and styling specific parts of the page

The Homepage

THe homepage is typically given the name index.html and serves as a place to put links to more detailed pages, logos, photos, and any other important information that you want to be visible at first glance.

Images

It is common to use images in your websites. There are some ways to do this to help organize and simplify the process. For example, make sure your images have clear and descriptive names as opposed to trying to link IMG29931. It is also common to have a folder called Images or images within your project folder, just to keep them organized and out of the way.
